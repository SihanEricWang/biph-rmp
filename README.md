# Rate My Teacher

## 项目简介

这是一个面向本高中学生的教师评价网站。  
学生可以在平台上查看教师信息并提交匿名评价，帮助大家了解课程体验与教学风格。

## 项目目标

1. 为学生提供一个反馈教师教学体验的平台  
2. 为选课和课程了解提供参考信息  
3. 鼓励理性和尊重的评价文化  

## 主要功能

1. 教师列表与教师信息展示  
2. 学生匿名评分与文字评论  
3. 教师与科目搜索功能  
4. 教师平均评分展示  

## 使用对象

本项目仅供本高中在校学生使用。

## 项目状态

项目处于早期开发阶段，功能仍在持续完善中。

## 未来计划

1. 添加用户登录系统  
2. 引入评论审核机制  
3. 优化界面与使用体验
